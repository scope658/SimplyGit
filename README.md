## SimplyGit
SimplyGit — это мобильный клиент для GitHub, разработанный на Kotlin Multiplatform.

## ⭐ Возможности
* **Авторизация:** Вход в систему через GitHub OAuth.
* **Онбординг:** Вводные экраны для новых пользователей.
* **Поиск:** Поиск репозиториев по названию на главном экране.
* **Детали репозитория:** Просмотр подробной информации о проекте.
* **Просмотр файлов:** Навигация по структуре файлов репозитория (списки папок и файлов).
* **Создание Issue:** Возможность создавать новые тикеты в репозиториях.
* **Профиль:** Отображение информации о пользователе.

## 🛠️ Технологический стек
- Kotlin Multiplatform (KMP)
- Compose Multiplatform (CMP)
- Koin
- Ktor
- Kotlinx Serialization
- Room
- AppAuth
- DataStore
- Coil
- Napier
- Kotlin Coroutines + Flow
- JUnit
- Compose UI Test (использование testTag)
